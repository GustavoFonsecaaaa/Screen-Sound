# Screen Sound 🎵

Projeto de console em C# para gerenciamento de bandas, desenvolvido para praticar lógica de programação, estruturas de repetição e manipulação de listas.

Este repositório faz parte dos meus estudos na trilha de C# e .NET da Alura.

---

## 🛠️ Tecnologias Utilizadas

* **C#** (Linguagem de programação)
* **.NET 8** (Plataforma de desenvolvimento)

## 📌 Funcionalidades Atuais

* [x] Exibição de mensagem de boas-vindas estilizada.
* [x] Menu interativo de opções de console.
* [ ] Cadastro de novas bandas *(em desenvolvimento)*.
* [ ] Avaliação e média de notas das bandas *(em desenvolvimento)*.
